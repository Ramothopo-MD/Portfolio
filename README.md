# 💼 Ramothopo Mosewa Desmond – Portfolio

A personal portfolio website built with **Spring Boot**, **Thymeleaf**, and **Tailwind CSS**, featuring a functional contact form with email integration.

---

## 🚀 Features

- Responsive modern UI (Tailwind CSS)
- Spring Boot backend
- Thymeleaf templating
- Contact form with email sending
- Environment variable support for secure credentials
- Clean MVC architecture

---

## 🛠 Tech Stack

- Java 17+
- Spring Boot
- Thymeleaf
- Maven
- Tailwind CSS
- Gmail SMTP (App Password)
